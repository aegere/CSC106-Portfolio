# CSC106 Portfolio Website

A responsive multi-page portfolio website developed as part of the **CSC106 – Introduction to Web Technologies** course.

🌐 https://aegere.github.io/CSC106-Portfolio/

## GitHub Repository

## Project Overview

This project demonstrates the application of HTML5, CSS3, JavaScript, Git, and GitHub Pages in building a responsive portfolio website.

The website includes:

- Home Page
- About Me Page
- Projects Page
- Academic Planner
- Contact Page

---

## Features

- Responsive website design
- Semantic HTML5 structure
- External CSS styling
- JavaScript interactivity
- Academic task planner

## Website Pages

### Home

Introduces visitors with a welcome message, professional profile photograph, and audio introduction.

### About

Displays educational background, professional certifications, technical skills, and career aspirations.

### Projects

Showcases sample data analytics projects with dashboard previews.

### Academic Planner

Allows users to:

- Add tasks
- Mark tasks as completed
- Delete tasks

### Contact

Provides a contact form with client-side validation.
